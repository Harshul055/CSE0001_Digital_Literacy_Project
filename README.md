# 🌐 Digital Literacy Portfolio

## 👋 About Me
Hello! I'm Harshul Agarwal, a first-year B.Tech student at VIT Bhopal University.  
This repository is part of my **CSE0001 – Digital Literacy** project, where I worked as a *Student Digital Ambassador* to explore essential digital skills required in today’s world.

---

## 📌 Project Overview
The goal of this project is to understand and apply key concepts of digital literacy, including:
- Building a professional online presence  
- Practising safe and responsible internet usage  
- Exploring coding and collaboration tools  
- Learning effective digital communication  
- Understanding cybercrime and prevention  

Each task in this repository reflects practical learning and real-world application.

## 🎯 Task Summaries

### 🟢 Task 1 – Digital Literacy Infographic
Created a visual infographic using Canva to explain digital literacy concepts such as safe internet practices, digital tools, and professional online presence.  
This task improved my ability to present information in a simple and engaging way.

---

### 🔵 Task 2 – Digital Portfolio
Built my professional presence on platforms like GitHub, LinkedIn, and Kaggle.  
This helped me understand the importance of maintaining a digital identity and how it can support future career opportunities.

---

### 🟡 Task 3 – Platforms Exploration
Solved a beginner coding problem on HackerRank and created a Digital Literacy Quiz using Google Forms.

## 🧠 Key Learnings
- Digital literacy is essential for both academic and professional growth  
- A strong online presence can create future opportunities  
- Safe internet practices are necessary to avoid cyber risks  
- Communication skills are just as important as technical skills  
- Practical tools like GitHub and Google Forms are very useful for students
